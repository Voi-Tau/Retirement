# Retirement
## 介绍
Retirement是一款以`星遥优化`整合包为底本、添加其他模组的休闲整合包。整合包聚焦农业与建筑（未来可能引入自动化），核心内容为农夫乐事系列。出于兼容考虑，未添加农夫乐事系列以外的农业模组。引用的模组详情请见`addmods.list`文件。整合包自`1.1.2`版本起使用本repo进行管理。出于引用模组的版权考虑，**无授权禁止使用**！
